# dev3000 References

- Repo: https://github.com/vercel-labs/dev3000

Notes:
- Use the upstream README for install/usage steps.
- Confirm any required env vars before running commands.

## Quick start

```bash
pnpm install -g dev3000
dev3000
```

Local install:

```bash
pnpm add -D dev3000
pnpm dev3000
```

## Options

- `--port <number>`: dev server port (default: 3000)
- `--mcp-port <number>`: MCP server port (default: 3684)
- `--disable-mcp-configs <targets>`: skip auto-writing MCP configs
- `--browser <path>`: use a specific Chromium browser
- `--servers-only`: skip browser monitoring
- `--headless`: headless mode (CI)

## Project notes

- Requires Node.js >= v22.12.0.
- Logs and timeline UI at `http://localhost:3684/logs`.
- README examples use `d3k` as a shorthand alias for `dev3000`.
